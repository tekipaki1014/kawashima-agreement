# SaaS開発業務委託契約書

合同会社ブリング（以下「甲」という。）と安藤（以下「乙」という。）は、甲が企画、開発、運営及び将来の譲渡、承継又は投資受入れを予定するSaaSサービスに関する開発業務の委託について、以下のとおり業務委託契約（以下「本契約」という。）を締結する。

## 第1条（目的）

本契約は、甲が企画、保有及び運営するSaaSサービスに関し、乙に対してシステム開発、改修、保守その他関連業務を委託するにあたり、権利関係、秘密保持、データ管理、引継ぎ及びその他必要事項を定め、当該SaaSサービスの継続運営及び将来の事業売却、会社売却、事業承継又は第三者へのライセンス等に支障が生じない状態を確保することを目的とする。

## 第2条（定義）

本契約において、次の各号の用語はそれぞれ以下の意味を有する。

1. **「本サービス」**とは、甲が企画、開発、運営又は提供するSaaSサービス及びこれに付随するアプリケーション、管理画面、API、データベース、サーバー設定、ドメイン、画面仕様、ドキュメントその他一切の関連資産をいう。
2. **「成果物」**とは、本業務に関連して乙が作成、修正、納入又は提供するプログラム、ソースコード、オブジェクトコード、仕様書、設計書、テスト仕様書、テスト結果、画面デザイン、UI、UX、マニュアル、データ定義書、インフラ設定情報、運用手順書、ドキュメントその他一切の有形無形の成果をいう。
3. **「乙既存資産」**とは、乙が本契約締結前から保有し又は第三者から適法に利用許諾を受けている汎用ライブラリ、テンプレート、ツール、ノウハウ、モジュール、コード片その他の資産をいう。
4. **「OSS」**とは、オープンソースソフトウェアをいう。
5. **「利用者データ」**とは、本サービスの利用者、見込客、管理者、取引先その他に関するデータ、ログ、設定情報、顧客情報、個人情報その他本サービスに関連して蓄積又は処理される一切のデータをいう。
6. **「対象アカウント等」**とは、ドメイン、クラウド環境、ソースコード管理環境、CI/CD、外部API、分析基盤、メール配信基盤、決済アカウント、監視ツール、証明書管理、バックアップ環境その他本サービスの開発、運営、保守及び継続利用に必要な一切のアカウント、契約、環境及び管理権限をいう。

## 第3条（本業務の内容）

1. 甲は乙に対し、次の業務その他個別契約、発注書、仕様書、タスク管理ツール、チャットその他電磁的方法により合意した業務（以下総称して「本業務」という。）を委託し、乙はこれを受託する。
   1. 要件整理、仕様調整、基本設計、詳細設計
   2. フロントエンド、バックエンド、API、バッチ、管理画面その他機能の開発及び改修
   3. データベース設計、インフラ構築、クラウド設定、セキュリティ設定
   4. テスト、不具合修正、性能改善、保守対応
   5. 成果物に関する説明資料、引継資料、運用資料の整備
   6. 将来の保守委託先、買主、承継先又は甲の指定する第三者への引継ぎ協力
2. 個別の作業内容、納期、報酬、作業範囲、検収条件等は、個別契約等で定める。
3. 本契約と個別契約等が矛盾抵触する場合、個別契約等が優先する。ただし、本契約における知的財産、秘密保持、個人情報、引継ぎ協力、権利承継及び事業売却対応に関する条項は、個別契約等に明示の反対規定がない限り優先適用される。

## 第4条（契約類型）

1. 本契約は原則として準委任契約とする。
2. ただし、個別契約等において成果物、納期、検収条件及び報酬が明確に定められた場合は、当該個別契約等に限り請負契約として取り扱う。
3. 乙は自己の裁量と責任において本業務を遂行し、甲は乙又は乙の従事者に対し、労働者派遣その他法令上問題となる直接具体的な指揮命令を行わない。

## 第5条（報酬）

1. 甲は乙に対し、本業務の対価として、個別契約等で定める報酬を支払う。
2. 乙は、毎月末日又は個別契約等で定める締日に請求書を発行し、甲は翌月末日までに乙指定口座へ振込送金により支払う。
3. 振込手数料は甲の負担とする。
4. 消費税等は別途甲の負担とする。

## 第6条（協力義務）

1. 甲は、乙による本業務遂行に必要な資料、仕様、判断、アカウント、テスト環境その他必要な情報を合理的な期間内に提供する。
2. 乙は、本業務に支障を生じさせるおそれのある事情、不具合、仕様上の疑義、セキュリティ上の懸念、権利侵害のおそれ、納期遅延のおそれ等を認識した場合、直ちに甲へ報告する。

## 第7条（再委託）

1. 乙は、甲の事前の書面又は電磁的方法による承諾なく、本業務の全部又は重要な一部を第三者に再委託してはならない。
2. 乙が再委託を行う場合、乙は再委託先に対して本契約と同等以上の義務を負わせ、再委託先の行為について一切の責任を負う。
3. 乙は、甲の求めがあるときは、再委託先の名称、所在地、担当範囲、秘密保持措置、セキュリティ管理体制その他甲が合理的に求める情報を開示する。

## 第8条（成果物の納入及び管理）

1. 乙は、請負型の個別契約等がある場合、成果物を個別契約等で定める方法により納入する。
2. 乙は、ソースコード、設計資料、環境設定情報、依存関係一覧、外部サービス接続情報一覧、データベース構成資料、運用手順、バックアップ手順、復旧手順、管理者権限情報一覧その他、甲又は甲の承継人が本サービスを継続運営するために合理的に必要な資料を、成果物の一部として整備するものとする。なお、これらに関連する元データ、管理画面、リポジトリ、クラウド設定及び各種アカウントは、すべて甲名義かつ甲管理のもとで保管及び運用されるものとする。
3. 乙は、甲の求めがある場合、Gitその他のバージョン管理システムにおいて、甲が管理主体となるアカウント、リポジトリ又は組織に成果物を格納するものとする。
4. 乙は、甲の求めがある場合、成果物及び関連資料を、第三者への引継ぎに支障のない一般的かつ合理的な状態で整理する。

## 第9条（検収）

1. 甲は、成果物の納入を受けた日から60日以内に検査を行い、その合否を乙に通知する。
2. 甲が前項期間内に合理的な理由を付した不合格通知をしない場合、当該期間の満了をもって検収合格とみなす。
3. 不合格の場合、乙は自己の責任において合理的期間内に修補又は追完を行い、再納入する。
4. 検収合格は、第17条に定める契約不適合責任その他乙の責任を免除するものではない。

## 第10条（知的財産権の帰属）

1. 成果物及び本業務の遂行過程で新たに作成又は発生した著作権（著作権法第27条及び第28条の権利を含む。）、特許を受ける権利、ノウハウその他一切の知的財産権は、発生と同時に甲に帰属するものとし、法令上又は性質上当然に乙に帰属する場合には、乙は甲に対し、成果物の完成及び当該成果物に係る報酬の支払完了を条件として、これを譲渡する。
2. 乙は、甲、甲の承継人、買主、事業譲受人、株式取得者、ライセンシー、保守委託先その他甲から権利を承継又は許諾された者に対し、成果物について著作者人格権を行使しない。
3. 乙は、成果物について、甲による改変、複製、翻案、再利用、第三者への譲渡、ライセンス、再委託保守、M&A、事業譲渡、会社分割、合併その他一切の処分又は利用を妨げない。
4. 乙既存資産の権利は乙に留保される。ただし、乙は甲に対し、甲及び甲の承継人、買主、保守委託先その他甲が指定する者が、目的を問わず自らの事業において利用、複製、翻案、改変等を行うため、乙既存資産を無償、永続的、取消不能な非独占的利用権として許諾する。なお、本利用権の譲渡又は再許諾は、本サービス又は本事業の譲渡、承継及び保守委託に伴う場合にのみ行うことができるものとする。
5. 乙は、成果物に乙既存資産又は第三者権利が含まれる場合、その範囲を事前に明示し、甲による本サービスの継続利用、改修、売却、承継又は第三者保守に支障がないよう必要な権利処理を完了させるものとする。
6. 乙は、甲の事前の書面承諾なく、成果物又は本サービスと実質的に同一のプログラム、仕様、画面、設計、顧客向け専用機能その他甲の競争優位に関わる部分を第三者へ提供してはならない。

## 第11条（第三者権利侵害の不存在等）

1. 乙は、成果物が第三者の著作権、特許権、商標権、営業秘密その他の権利を侵害しないよう合理的な注意を尽くす。
2. 乙は、第三者との間で成果物又は本サービスに関し権利侵害の疑義又は紛争が生じた場合、直ちに甲に通知し、自己の責任と費用で解決に努める。
3. 前項の場合、乙は甲の事業継続又は売却手続に重大な支障が出ないよう、代替措置、修正、権利取得その他合理的措置を講じる。

## 第12条（OSS及び第三者ソフトウェア）

1. 乙は、OSS又は第三者ソフトウェアを利用する場合、その名称、バージョン、ライセンス名、利用箇所、利用態様及び甲の事業売却、継続利用又は再ライセンスへの影響の有無を一覧化し、甲の求めに応じて提出する。
2. 乙は、甲の事前承諾なく、ソースコード開示義務、頒布条件、逆ライセンス義務その他甲の本サービスの独占的利用、事業売却、ライセンス提供又は第三者保守に重大な支障を与える条件が付されたOSS又は第三者ソフトウェアを組み込んではならない。
3. 乙は、利用したOSS及び第三者ソフトウェアについて、適用ライセンスを遵守し、必要な表示、通知、ソース提供義務その他の対応を行う。
4. 乙は、OSS又は第三者ソフトウェアに重大な脆弱性、ライセンス違反又は利用制限のおそれを認識した場合、直ちに甲へ報告し、合理的な是正案を提示する。

## 第13条（アカウント、インフラ及び外部サービス）

1. 対象アカウント等は、すべて甲名義とし、かつ甲が最終的かつ完全な管理権限を有するものとする。
2. 乙は、対象アカウント等について、自己名義、自己管理又は自己を主たる管理者とする状態を作ってはならず、またこれを維持してはならない。
3. 乙は、本業務遂行上必要な範囲で甲から対象アカウント等の利用許可を受けることができるが、その場合であっても、当該権限は甲から一時的に付与される補助的権限にとどまり、対象アカウント等に関する権利、名義、管理権限その他一切の支配権は甲に帰属する。
4. 乙は、対象アカウント等に関し、甲の事前の書面又は電磁的方法による承諾なく、新規契約、設定変更、権限追加、外部サービス連携、仕様変更その他本サービスの継続運営又は事業売却に影響を及ぼし得る行為をしてはならない。

7. 乙が本条に違反して自己名義又は自己管理のアカウント、契約又は環境を本サービスに組み込んだ場合、乙は自己の責任と費用において、甲の指定する期限までにこれを甲名義かつ甲管理の状態へ完全に移行しなければならない。

## 第14条（秘密保持）

1. 甲及び乙は、本契約又は本業務に関連して相手方から開示を受け又は知り得た技術上、営業上、経営上その他一切の非公知情報を秘密情報として厳重に管理し、相手方の事前承諾なく第三者へ開示又は漏えいしてはならず、本契約の目的以外に使用してはならない。
2. 次の情報は秘密情報に含まれない。
   1. 開示時に公知であった情報
   2. 開示後、受領者の責によらず公知となった情報
   3. 開示前から適法に保有していた情報
   4. 正当な権限を有する第三者から適法に取得した情報
   5. 秘密情報によらず独自に取得又は開発した情報
3. 乙は、甲が将来、本サービス又は関連事業について売却、資本提携、資金調達、デューデリジェンスその他これらに準ずる手続を行う場合、甲が秘密保持契約を締結した候補先、専門家、金融機関、買主候補、投資家候補等に対して必要な範囲で本契約、成果物、仕様、運用情報その他必要資料を開示することにあらかじめ同意する。

5. 本条の義務は、本契約終了後も期間の定めなく存続する。

## 第15条（個人情報及び利用者データ）

1. 乙は、個人情報、個人データ及び利用者データについて、法令、ガイドライン及び甲の指示に従い、本業務遂行に必要な範囲でのみ取り扱う。
2. 乙は、前項のデータを自己又は第三者のために利用し、分析し、持ち出し、複製し、保存し、第三者提供し、又は学習データその他二次利用の目的に供してはならない。ただし、甲の事前承諾がある場合を除く。
3. 乙は、甲の求めに応じ、個人情報及び利用者データの取扱状況、保存場所、アクセス権限、安全管理措置、再委託状況その他合理的に必要な事項を報告する。

6. 利用者データに関する権利及び管理権限は、甲に帰属する。

## 第16条（セキュリティ）

1. 乙は、本業務遂行にあたり、合理的な安全管理措置を講じる。
2. 乙は、アクセス制御、認証情報管理、マルウェア対策、脆弱性管理、ログ管理、バックアップ、端末管理その他必要な措置を実施する。
3. 乙は、甲の事前承諾なく、本番データをローカル環境又は私的環境へ複製してはならない。
4. 乙は、甲の求めに応じ、セキュリティに関する説明又は改善提案を行う。

## 第17条（契約不適合及び不具合対応）

1. 請負型個別契約において成果物が仕様に適合しない場合、乙は自己の責任と費用で修補、追完又は代替対応を行う。
2. 甲は、検収後1年以内に契約不適合を通知した場合、乙に対し前項の対応を求めることができる。ただし、乙の故意又は重大な過失による場合はこの限りでない。
3. 本番障害、重大不具合、セキュリティ上の問題又は第三者権利侵害のおそれが発生した場合、乙は、甲のサービス継続及び売却手続への影響を最小化するよう合理的に協力する。

## 第18条（引継ぎ協力義務）

1. 本契約終了時、個別契約終了時、又は甲が本サービス若しくは関連事業の売却、承継、組織再編、保守会社変更その他合理的必要性を有する場合、乙は、甲又は甲の指定する第三者に対し、以下の引継ぎ協力を行う。
   1. ソースコード、仕様書、設計書、設定情報、運用資料その他必要資料の引渡し
   2. システム構成、依存関係、外部サービス、障害履歴、運用上の注意点等の説明
   3. 甲名義かつ甲管理で保有されるアカウント、権限、鍵、証明書、ドメイン、クラウド設定、外部サービス契約等の確認、整理及び移管協力
   4. 質疑応答、打合せ参加、保守委託先又は買主候補への説明
   5. その他円滑な事業承継又は運営移管に合理的に必要な事項
2. 前項のうち通常の業務終了に伴う最低限の引継ぎは本契約の報酬に含むものとし、それを超える相当量の作業については、甲乙協議のうえ別途有償とする。
3. 乙は、引継ぎ協力を拒み、妨げ、又は不当に遅延させてはならない。

## 第19条（権利義務の承継）

1. 甲は、事業譲渡、会社分割、合併、株式譲渡、持株会社化、第三者への事業承継その他組織再編又はM&Aを行う場合、あらかじめ乙に通知し、協議のうえ承継する。
2. 乙は、甲の承継先に対し、本契約と同条件で引き続き必要な協力を行うものとする。
3. 乙は、甲の事前承諾なく、本契約上の地位又は権利義務を第三者へ譲渡し、承継させ、又は担保設定してはならない。

## 第20条（表明保証）
